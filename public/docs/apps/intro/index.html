Aplicaciones - Introducción

Aplicaciones

Las aplicaciones en la plataforma utilizan el lenguaje QML desarrollado por Qt por lo que muchos de los tipos de datos que vienen con Qt se pueden utilizar para las mismas. Sin embargo, existen algunos tipos de datos desarrollados para ser utilizados con la plataforma. Ese es el caso del cliente de comunicaciones compatible con MQTT y del elemento Gauge que ha sido recreado para no interferir con los otros elementos.

Contenido

El contenido de la documentación de aplicaciones inluye los siguiente puntos:

  • Editor: En ésta sección se da una descripción del editor y su uso de forma resumida
  • Layout: En ésta sección se da una descripción de la API de Qt para realizar Layouts al igual que algunos componentes de uso común
  • Comunicación: En ésta sección se da una descripción de la API desarrollada para la comunicación con plataforma mediante WebSockets
