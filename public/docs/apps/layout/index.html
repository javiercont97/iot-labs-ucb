Aplicaciones - Layout

Layout

Los layouts en QML se pueden construir de muchas formas, pero la más simple es utilizando los componentes Column y Row.
La forma de utilizar ambos es muy similar dado que comparten la misma API:

import QtQuick 2.12
import QtQuick.Controls 2.12

Item {
id: root
Column {
spacing: 5 // spacing determina la separación entre componentes
width: 300 // width determina el ancho de la columna, para esto es posible utilizar anchors.
height: 300 // height determina el alto de la columna, para esto es posible utilizar anchors.
Label {
// Propiedades de Label
}
Button {
// Propiedades de Button
}
Row {
// Es posible insertar elementos de layout dentro de otros
}
}

Row {
spacing: 5 // spacing determina la separación entre componentes
width: 300 // width determina el ancho de la fila, para esto es posible utilizar anchors.
height: 300 // height determina el alto de la fila, para esto es posible utilizar anchors.

Label {
// Propiedades de Label
}
Button {
// Propiedades de Button
}
Column {
// Es posible insertar elementos de layout dentro de otros
}
}
Row {
anchors.fill: parent // es posible utilizar anchors para algunas propiedades. En este caso la fila ocupará todo el espacio (fill) de su elemento padre, id: root
}
}
Elementos de layout

Controles

Existen diversos controles que se pueden utilizar para construir interfaces gráficas. Aquí veremos solo los más relevantes.

import QtQuick 2.12
import QtQuick.Controls 2.12

Item {
id: root
Column {
anchors.centerIn: parent // Colocar la columna en el centro del elemento padre
spacing: 50
Label {
text: "Hello world"
font.pointSize: 25 // Cambiar el tamaño de la fuente a 25pt
}
Button {
text: "Click me!"
onClicked: {
// Función de JavaScript que se ejecuta cuando el boton es clickeado
// En esta función no se reciben argumentos. Sin embargo, es posible manipular otros elementos mediante el id
}
}
Switch {
text: "Switch me!"
onCheckedChanged: {
// Función de JavaScript que se ejecuta cuando el switch cambia de estado
// En esta función se recibe el argumento checked. También es posible manipular otros elementos mediante el id
if (checked) {
// El switch esta encendido
} else {
// El switch esta apagado
}
}
}
Slider {
from: 0 // Valor minimo. El valor por omisión es 0
to: 100 // Valor máximo. El valor por omisión es 1
stepSize: 2 // Valor de los incrementos
orientation: Qt.Vertical // El valor por omisión es Qt.Horizontal. Los valores posibles son Qt.Vertical y Qt.Horizontal
onValueChanged: {
// Función de JavaScript que se ejecuta cuando el valor del Slider cambia
// Esta funcion recibe el argumento value
console.log ( value ); // Mostrar el valor del Slider en la consola del navegador
}
}
}
}

Controles avanzados

import QtQuick 2.12
import QtQuick.Controls 2.12
import Controls 1.0 // Importar controles avanzados

Item {
id: root
Column {
anchors.centerIn: parent // Colocar la columna en el centro del elemento padre
spacing: 50
Dial {
id: dial // Asignar el id dial
from: 0 // Valor minimo. El valor por omisión es 0
to: 100 // Valor máximo. El valor por omisión es 1
stepSize: 2 // Valor de los incrementos
onValueChanged: {
// Función de JavaScript que se ejecuta cuando el valor del Dial cambia
// Esta funcion recibe el argumento value
console.log ( value ); // Mostrar el valor del Dial en la consola del navegador
gauge.value = value; // Cambiar el valor de gauge con el valor del Dial
}
}
Gauge {
id: gauge // Asignar el id gauge
width: 200 // Determinar el ancho del gauge
height: 200 // Determinar el alto del gauge
min: 0 // Valor minimo. El valor por omisión es 0
max: 100 // Valor máximo. El valor por omisión es 100
}
Gauge {
width: 200 // Determinar el ancho del gauge
height: 200 // Determinar el alto del gauge
min: 0 // Valor minimo. El valor por omisión es 0
max: 100 // Valor máximo. El valor por omisión es 100
value: dial.value // Utilizar propery binding para asociar el valor del gauge con el de dial
}
}
}
