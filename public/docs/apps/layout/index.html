Aplicaciones - Layout

Layout

Los layouts en QML se pueden construir de muchas formas, pero la más simple es utilizando los componentes Column y Row.
La forma de utilizar ambos es muy similar dado que comparten la misma API:

1import QtQuick 2.12
2import QtQuick.Controls 2.12
3
4Item {
5 id: root
6 Column {
7 spacing: 5 // spacing determina la separación entre componentes
8 width: 300 // width determina el ancho de la columna, para esto es posible utilizar anchors.
9 height: 300 // height determina el alto de la columna, para esto es posible utilizar anchors.
10 Label {
11 // Propiedades de Label
12 }
13 Button {
14 // Propiedades de Button
15 }
16 Row {
17 // Es posible insertar elementos de layout dentro de otros
18 }
19 }
20
21 Row {
22 spacing: 5 // spacing determina la separación entre componentes
23 width: 300 // width determina el ancho de la fila, para esto es posible utilizar anchors.
24 height: 300 // height determina el alto de la fila, para esto es posible utilizar anchors.
25
26 Label {
27 // Propiedades de Label
28 }
29 Button {
30 // Propiedades de Button
31 }
32 Column {
33 // Es posible insertar elementos de layout dentro de otros
34 }
35 }
36
37 Row {
38 anchors.fill: parent // es posible utilizar anchors para algunas propiedades. En este caso la fila ocupará todo el espacio (fill) de su elemento padre, id: root
39 }
40}
Elementos de layout

Controles

Existen diversos controles que se pueden utilizar para construir interfaces gráficas. Aquí veremos solo los más relevantes.

1import QtQuick 2.12
2import QtQuick.Controls 2.12
3
4Item {
5 id: root
6 Column {
7 anchors.centerIn: parent // Colocar la columna en el centro del elemento padre
8 spacing: 50
9 Label {
10 text: "Hello world"
11 font.pointSize: 25 // Cambiar el tamaño de la fuente a 25pt
12
13 }
14 Button {
15 text: "Click me!"
16 onClicked: {
17 // Función de JavaScript que se ejecuta cuando el botón es clickeado
18 // En esta función no se reciben argumentos. Sin embargo, es posible manipular otros elementos mediante el id
19 }
20 }
21 Switch {
22 text: "Switch me!"
23 onCheckedChanged: {
24 // Función de JavaScript que se ejecuta cuando el switch cambia de estado
25 // En esta función se recibe el argumento checked. También es posible manipular otros elementos mediante el id
26 if (checked) {
27 // El switch esta encendido
28 } else {
29 // El switch esta apagado
30 }
31 }
32 }
33 Slider {
34 from: 0 // Valor minimo. El valor por omisión es 0
35 to: 100 // Valor máximo. El valor por omisión es 1
36 stepSize: 2 // Valor de los incrementos
37 orientation: Qt.Vertical // El valor por omisión es Qt.Horizontal. Los valores posibles son Qt.Vertical y Qt.Horizontal
38 onValueChanged: {
39 // Función de JavaScript que se ejecuta cuando el valor del Slider cambia
40 // Esta funcion recibe el argumento value
41 console.log ( value ); // Mostrar el valor del Slider en la consola del navegador
42 }
43 }
44 }
45}
