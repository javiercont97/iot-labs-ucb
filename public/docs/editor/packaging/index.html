Editor - Empaquetado

Empaquetado y Distribución

Para empaquetar y distribuir las aplicaciones por internet existe una herramienta en el editor. Cuando el usuario desee publicar la aplicación es suficiente con utilizar la herramienta y cargar el paquete generado a la plataforma. El tiempo que demora la compilación de la aplicación en una página web se encuentra entre 2 y 10 minutos. Cuando su aplicación este lista recibirá un correo electrónico como notificación.

Archivos generados

El proceso de empaquetado genera dos archivos. Un archivo llamado userApp.qml y otro con extensión .zip. El archivo ZIP es el paquete que contiene la aplicación y debe ser cargado en la plataforma. Mientras que el archivo QML es un archivo temporal que puede ser eliminado sin riesgo.

Cargando archivo a la plataforma

Para cargar el paquete generado debe dirigirse a la página de aplicaciones y presionar el tercer botón de la tarjeta de la aplicación que desea cargar.

El siguiente paso es cargar el archivo ZIP generado mediante la ventana modal de carga de archivos.

Finalmente debe esperar a que termine el proceso de compilación, el servidor enviará un correo electrónico notificando que el proceso ha terminado.
