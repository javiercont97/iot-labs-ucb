Message Logger - Introducción

Message Logger

La aplicación Message Logger ha sido desarrollada en el marco del proyecto es un cliente MQTT con interfaz gráfica que integra una función para registrar los mensajes recibidos en un archivo CSV además de proporcionar un publicador que facilitará la tarea en el desarrollo para el estudiante. Ya que podrá publicar mensajes a la interfaz o al dispositivo sin tener ambos. Para la herramienta al igual que el editor descargue el instalador unificado proporcionado en la página de descargas.
